# Product

## Register

product

## Users

Compyle Code is for developers who want a privacy-first, AI-native editor that feels cohesive instead of like a loose collection of VS Code panels. Users are building, debugging, converting, and shipping projects inside a desktop IDE, often with AI help and local tooling in the loop.

## Product Purpose

Compyle Code helps developers turn ideas into runnable projects, keep project context organized, and use AI or local models without being forced through a cloud backend. Success means the editor feels premium, trustworthy, fast, and clear during real coding work.

## Brand Personality

Premium calm, precise, privacy-first.

## Anti-references

Do not look like a cheap VS Code fork: avoid bolted-on panels, mismatched spacing, vague AI labels, uneven split panes, decorative glass, and half-finished controls. Do not clone Apple UI directly; use Liquid Glass principles in a Compyle-native workbench language.

## Design Principles

- The code and task stay primary; visual effects support focus.
- Every Compyle surface should feel like one product, not separate extensions.
- Privacy choices must be visible, understandable, and reversible.
- AI workflows should be reviewable before they modify project files.
- Strong defaults matter: users should not need to tune the app before it feels credible.

## Accessibility & Inclusion

Target WCAG AA contrast for readable text and controls. Respect reduced motion, reduce transparency, and Safe Mode as hard overrides. Liquid Glass must fail back to readable opaque surfaces when performance, transparency, or contrast would be poor.
