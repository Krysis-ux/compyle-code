## Telemetry

Compyle Code does **not** send telemetry to Microsoft by default. Telemetry is disabled by default. If Compyle telemetry is added in the future, it will be opt-in only with clear user notification.

---

## AI Features

Compyle Brain (Compyle's AI assistant) connects only to AI providers configured by the user. No code or data is sent to any cloud AI service without explicit user action. Users supply their own API keys; Compyle does not operate a mandatory AI backend.
